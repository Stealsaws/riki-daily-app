เชื่อมต่อ Google Apps Script

❌ เชื่อมต่อไม่ได้ — ตรวจสอบ URL และการ Deploy
วิธีรับ URL (ทำครั้งเดียว):
1. เปิด script.google.com → New Project
2. วางโค้ดจากไฟล์ riki_apps_script.js
3. แก้ SHEET_ID และ FOLDER_ID
4. Run → initializeSheets (ครั้งแรก)
5. Deploy → New Deployment → Web App
  • Execute as: Me
  • Who has access: Anyone
6. คัดลอก URL มาวางด้านบน
